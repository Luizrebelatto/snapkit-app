# App Snapkit

## Sobre
App desenvolvido com snapkit, view code, constraints e swift

## Tecnologias
- Swift
- View Code
- SnapKit ~> 5.7.0
- Xcode ~> 15.4

## Como executar

```
git clone https://github.com/Luizrebelatto/snapkit-app.git
```

```
cd snapkit-app && pod install
```
